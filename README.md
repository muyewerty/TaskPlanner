# TaskPlanner

Компактный, интуитивно понятный и кроссплатформенный планировщик задач, написанный на Python с использованием фреймворка Kivy. 

Проект разработан для эффективного управления личным временем, структурирования списков дел и контроля за дедлайнами с помощью системы уведомлений.

# 🌟 Особенности

* **Создание, редактирование и удаление задач.**

* **Офлайн-работа:** Все данные сохраняются локально в формате JSON, без привязки к облаку.

* **Настройка напоминаний:** Возможность установить таймер, который пришлет системное уведомление благодаря фоновым потокам.

* **Кроссплатформенность:** Благодаря фреймворку Kivy, приложение работает на Windows, macOS, Linux и может быть портировано на мобильные устройства.

# 🛠 Технологии
* **Python 3.x**
* **Kivy** — построение графического интерфейса пользователя.
* **Plyer** — кроссплатформенный механизм отображения системных уведомлений.
* **Модули ОС:** `json`, `os`, `threading`, `time`.

# Установка

* **cd TaskPlanner**

* **pip install -r requirements.txt**
